# JavaScript Learning Journey 🚀

Welcome to your comprehensive guide to mastering JavaScript! This repository contains a structured curriculum taking you from absolute beginner to advanced concepts and TypeScript.

## 📂 Modules Overview

### Foundations
- **[01-Basics](./01-Basics/)**: Variables, Types, Operators.
- **[02-Control-Flow](./02-Control-Flow/)**: Loops, Conditionals, Logic.
- **[03-Functions](./03-Functions/)**: Arrow functions, IIFE, Closures.
- **[04-Data-Structures](./04-Data-Structures/)**: Arrays, Maps, Sets, Objects.

### Core DOM & Logic
- **[05-DOM-Manipulation](./05-DOM-Manipulation/)**: Event Listeners, Observers.
- **[06-Advanced](./06-Advanced/)**: 'this', Hoisting, Call Stack, Memory.
- **[07-Async-JS](./07-Async-JS/)**: Promises, Async/Await, Event Loop.

### Modern Development
- **[08-Modern-OOP](./08-Modern-OOP/)**: Classes, Inheritance, Private Fields.
- **[09-Error-Handling](./09-Error-Handling/)**: Try/Catch, Custom Errors.
- **[10-Modern-JS-Features](./10-Modern-JS-Features/)**: Destructuring, Optional Chaining.
- **[11-Iterators-and-Generators](./11-Iterators-and-Generators/)**: Generators, Custom Iterators.
- **[12-Regular-Expressions](./12-Regular-Expressions/)**: Regex patterns & methods.

### Professional Skills
- **[13-Web-APIs](./13-Web-APIs/)**: Fetch, Storage, URL.
- **[14-Functional-Programming](./14-Functional-Programming/)**: Pure Functions, Currying.
- **[15-Modules-In-Depth](./15-Modules-In-Depth/)**: ESM, CommonJS, Bundlers.
- **[16-Testing](./16-Testing/)**: Unit Testing, TDD concepts.
- **[17-TypeScript-Intro](./17-TypeScript-Intro/)**: Typing, Interfaces, Generics.

## 🎓 How to use this repo
1. Open a module folder.
2. Read `theory.md` to understand the concepts.
3. Open `script.js` (or `.ts`) to read the code.
4. Run the code! You can run most files using Node.js:
   ```bash
   node 01-Basics/script.js
   ```
   *For DOM modules (05, 13), open `index.html` in your browser.*

Happy Coding! 💻
